
# 🚘 Waymo Open Dataset Perception Analytics

A futuristic autonomous driving analytics dashboard built using React, TypeScript, Framer Motion, and LiDAR perception metrics.

This platform analyzes how self-driving vehicles detect and understand objects such as vehicles, pedestrians, cyclists, and road infrastructure using perception intelligence and real-world driving analytics.

# 🧠 Project Overview

Self-driving vehicles use:

- **Cameras** → capture visual road information
- **LiDAR** → generate 3D spatial understanding using laser reflection

This project simulates how autonomous vehicle systems evaluate object detection performance across different road environments and driving conditions.

The dashboard provides a performance report card for:

- Vehicles
- Pedestrians
- Cyclists
- Traffic infrastructure

---

# 📊 Analytics Included

| Metric | Description |
|---|---|
| Precision | Accuracy of detected objects |
| Recall | Ability to detect all real objects |
| F1 Score | Combined detection performance |
| Detection Accuracy | Overall object recognition effectiveness |
| AI Failure Insights | Weak perception zones and risks |

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/waymo-perception-analytics.git
```

## Navigate to Project

```bash
cd waymo-perception-analytics
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
